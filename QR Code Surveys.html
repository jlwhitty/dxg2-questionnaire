📊 Google Sheets Setup

  1. Create a Google Sheet (one tab per survey, or two separate sheets)
  2. Add column headers matching the survey fields (see comments in each HTML file)
  3. Go to Extensions → Apps Script
  4. Paste the doPost() function from the HTML file comments
  5. Click Deploy → New Deployment → Web App
  6. Set "Who has access" to Anyone
  7. Copy the deployment URL
  8. Paste it into the GOOGLE_SCRIPT_URL variable in each survey HTML file
